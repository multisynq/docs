## Installation

Get started with Multisynq by installing the SDK and setting up your first application.

### Install the SDK

<AccordionGroup>
  <Accordion icon="npm" title="Install via npm">
    Install the Multisynq SDK using npm:
    ```bash
    npm install @multisynq/sdk
    ```
  </Accordion>
  <Accordion icon="code" title="Basic Setup">
    Create a new Multisynq application:
    ```javascript
    import { Multisynq } from '@multisynq/sdk';

    const app = new Multisynq({
      apiKey: 'your-api-key',
      appId: 'your-app-id'
    });

    app.connect();
    ```
  </Accordion>
</AccordionGroup>

### Your First Collaborative App

<AccordionGroup>

<Accordion icon="users" title="Create a shared session">
  Set up a shared session where multiple users can collaborate:
  ```javascript
  const session = await app.createSession({
    name: 'my-first-session',
    maxUsers: 10
  });
  ```
</Accordion>

<Accordion icon="sync" title="Synchronize state">
  Share state between users in real-time:
  ```javascript
  // Set shared state
  session.setState('counter', 0);

  // Listen for state changes
  session.on('stateChange', (key, value) => {
    console.log(`${key} changed to ${value}`);
  });

  // Update state (syncs across all users)
  session.setState('counter', 42);
  ```
</Accordion>

</AccordionGroup>
