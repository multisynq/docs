---
title: "Delete Session"
api: "DELETE https://api.multisynq.io/v1/sessions/{id}"
description: "This endpoint deletes a collaboration session."
---

This endpoint deletes a collaboration session and disconnects all participants.

## Request

<ParamField path="id" type="string" required>
  The unique identifier of the session to delete
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication
</ParamField>

<ParamField query="force" type="boolean">
  Force deletion even if participants are still connected (default: false)
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier of the deleted session
</ResponseField>

<ResponseField name="deleted" type="boolean">
  Confirmation that the session was deleted
</ResponseField>

<ResponseField name="deletedAt" type="string">
  ISO timestamp when the session was deleted
</ResponseField>

<ResponseField name="participantsDisconnected" type="number">
  Number of participants that were disconnected
</ResponseField>

<ResponseExample>

```json Response
{
  "id": "session_abc123",
  "deleted": true,
  "deletedAt": "2024-01-15T11:00:00Z",
  "participantsDisconnected": 2
}
```

</ResponseExample>
