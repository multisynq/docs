---
title: "Get Session"
api: "GET https://api.multisynq.io/v1/sessions/{id}"
description: "This endpoint retrieves a collaboration session by ID."
---

This endpoint returns a collaboration session by its ID, including current state, participants, and metadata.

## Request

<ParamField path="id" type="string" required>
  The unique identifier of the session to retrieve
</ParamField>

<ParamField header="Authorization" type="string" required>
  Bearer token for API authentication
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier of the session
</ResponseField>

<ResponseField name="appId" type="string">
  The application ID this session belongs to
</ResponseField>

<ResponseField name="state" type="object">
  The current synchronized state of the session
</ResponseField>

<ResponseField name="participants" type="array">
  Array of users currently in the session
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO timestamp when the session was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO timestamp when the session was last updated
</ResponseField>

<ResponseField name="metadata" type="object">
  Additional session metadata and configuration
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Whether the session is currently active
</ResponseField>
