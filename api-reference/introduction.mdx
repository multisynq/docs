## Welcome to Multisynq API

The Multisynq API provides everything you need to build real-time collaborative applications. Our REST API and WebSocket connections enable seamless synchronization between multiple users and devices.

## Base URL

All API requests should be made to:
```
https://api.multisynq.io/v1
```

## Authentication

All API endpoints require authentication using your API key. Include your API key in the Authorization header:

```json
{
  "Authorization": "Bearer YOUR_API_KEY"
}
```

You can obtain your API key from the [Multisynq Dashboard](https://dashboard.multisynq.io).

## Rate Limits

Our API has the following rate limits:
- **Free tier**: 1,000 requests per hour
- **Pro tier**: 10,000 requests per hour
- **Enterprise**: Custom limits

## WebSocket Connection

For real-time features, connect to our WebSocket endpoint:
```
wss://api.multisynq.io/v1/ws
```

## SDKs

We provide official SDKs for popular programming languages:

- **JavaScript/TypeScript**: `@multisynq/sdk`
- **React**: `@multisynq/react`
- **Vue**: `@multisynq/vue`
- **Python**: `multisynq-python`
- **Go**: `multisynq-go`
