### Instance Methods

#### `has(key, item, defaultValue)`

- has("debug", "recv", false) matches debug=recv and debug=send,recv - has("debug", "recv", true) matches debug=norecv and debug=send,norecv - has("debug", "recv", "localhost") defaults to true on localhost, false otherwise

**Parameters:**

- `key` (`String`) - key for list of items
- `item` (`String`) - value to look for in list of items
- `defaultValue` (`Boolean|String`) - if string, true on that hostname, false otherwise
