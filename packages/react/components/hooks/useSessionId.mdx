## useSessionId

<a id="usesessionid"></a>
<p>Hook that returns the ID of the current Multisynq session.</p>
### Syntax
```typescript
function useSessionId(): string | null
```
### Returns
<ResponseField type="string | null">
<p>The ID of the current session, or null if not in a session.</p>
</ResponseField>
### Examples
<CodeGroup>
```javascript
const sessionId = useSessionId();
console.log(sessionId ? `Current session: ${sessionId}` : 'Not in a session');
```
</CodeGroup>
