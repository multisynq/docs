## useView

<a id="useview"></a>
<p>Hook that provides access to the current Multisynq React View.</p>
### Syntax
```typescript
function useView(): MultisynqReactView<M> | null
```
### Returns
<ResponseField type="MultisynqReactView<M> | null">
<p>The current Multisynq React View, or null if not available.</p>
</ResponseField>
### Examples
<CodeGroup>
```javascript
const view = useView<GameView>();
if (view) {
  console.log(`Current view ID: ${view.viewId}`);
}
```
</CodeGroup>
