## useUpdateCallback

<a id="useupdatecallback"></a>
<p>Hook that sets up a callback for Multisynq.View.update(). The callback function is called at each simulation cycle.</p>
### Syntax
```typescript
function useUpdateCallback(callback: function(): void): void
```
### Parameters
<ParamField>
<ParamField path="callback" type="function(): void" required={true}>
<p>The function to be called at each simulation cycle</p>
</ParamField>
</ParamField>
### Examples
<CodeGroup>
```javascript
useUpdateCallback((update_time: number) => console.log(`Updated at ${update_time}!`));
```
</CodeGroup>
