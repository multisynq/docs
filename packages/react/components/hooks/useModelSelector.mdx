## useModelSelector

<a id="usemodelselector"></a>
<p>Hook that selects and returns a specific part of the Model state. This hook updates state only when the selected data changes, using hashing to detect deep equality. This hook is a good alternative to <a href="#usereactmodelroot">useReactModelRoot</a>, since it only re-renders when the subscribed part of the model changes. The given <code>selector</code> function should not return <code>undefined</code>.</p>
### Syntax
```typescript
function useModelSelector(selector: function(M): R): R | null
```
### Parameters
<ParamField>
<ParamField path="selector" type="function(M): R" required={true}>
<p>A function that selects data from the model. The return type of this function will be used as the return type of the hook.</p>
</ParamField>
</ParamField>
### Returns
<ResponseField type="R | null">
<p>The selected part of the Model state, or null if there is no current session.</p>
</ResponseField>
### Examples
<CodeGroup>
```javascript
const playerScore = useModelSelector((model: GameModel) => model.playerScore);
if (playerScore !== null) {
  console.log(`Current score: ${playerScore}`);
}
```
</CodeGroup>
