## useMultisynqSession

<a id="usemultisynqsession"></a>
<p>Hook that returns the current Multisynq session.</p>
### Syntax
```typescript
function useMultisynqSession(): MultisynqSession<MultisynqReactView<M>> | null
```
### Returns
<ResponseField type="MultisynqSession<MultisynqReactView<M>> | null">
<p>The current Multisynq session, or null if not in a session.</p>
</ResponseField>
### Examples
<CodeGroup>
```javascript
const session = useSession<GameModel>();
if (session) {
  console.log(`Connected to session: ${session.id}`);
}
```
</CodeGroup>
