## useSubscribe

<a id="usesubscribe"></a>
<p>A hook to set up a subscription to a Multisynq message.</p>
### Syntax
```typescript
function useSubscribe(scope: string, eventSpec: string, callback: function(T): void): void
```
### Parameters
<ParamField>
<ParamField path="scope" type="string" required={true}>
<p>The scope in which we are subscribing to the event</p>
</ParamField>
<ParamField path="eventSpec" type="string" required={true}>
<p>The name of the event we are subscribing to</p>
</ParamField>
<ParamField path="callback" type="function(T): void" required={true}>
<p>The function to be called when the event is received</p>
</ParamField>
</ParamField>
### Examples
<CodeGroup>
```javascript
function grabBall(data:GrabData): void  {
 // Callback logic here
}
useSubscribe<GrabData>(model.id, "grabbed", grabBall);
```
</CodeGroup>
