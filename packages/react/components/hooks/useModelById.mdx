## useModelById

<a id="usemodelbyid"></a>
<p>Hook that returns a <em>reference</em> to the Model instance with the given id. Keep in mind that even if the model data changes, React will not rerender the components that depend on its data. To achieve this behavior use <a href="#usereactmodelroot">useReactModelRoot</a> or <a href="#usemodelselector">useModelSelector</a> instead.</p>
### Syntax
```typescript
function useModelById(id: string): any
```
### Parameters
<ParamField>
<ParamField path="id" type="string" required={true}>
<p>The id of the Model to retrieve.</p>
</ParamField>
</ParamField>
### Returns
<ResponseField type="any">
<p>{M | null} The model instance with the given id. Returns <code>null</code> if not currently joined to any session, or if an image with the given id was not found</p>
</ResponseField>
### Examples
<CodeGroup>
```javascript
const userModel = useModelById<UserModel>('user-123');
if (userModel) {
  console.log(userModel.name);
}
```
</CodeGroup>
