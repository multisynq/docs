## useDetachCallback

<a id="usedetachcallback"></a>
<p>Hook that sets up a callback for Multisynq.View.detach(). The callback function is called when the root View is detached.</p>
### Syntax
```typescript
function useDetachCallback(callback: function(): void): void
```
### Parameters
<ParamField>
<ParamField path="callback" type="function(): void" required={true}>
<p>The function to be called when the root View is detached</p>
</ParamField>
</ParamField>
### Examples
<CodeGroup>
```javascript
function onDetached(): void  {
 // Callback logic here
}
useDetachCallback(onDetached);
```
</CodeGroup>
