## useLeaveSession

<a id="useleavesession"></a>
<p>Hook that provides a function to leave the current Multisynq session. Calling the returned function does not affect the <code>sessionParams</code> stored in <a href="#multisynqroot">MultisynqRoot</a>.</p>
### Syntax
```typescript
function useLeaveSession(): Function
```
### Returns
<ResponseField type="Function">
<p>A function that, when called, will leave the current Multisynq session.</p>
</ResponseField>
### Examples
<CodeGroup>
```javascript
const leaveSession = useLeaveSession();
// Later in your code:
leaveSession();
```
</CodeGroup>
