## usePublish

<a id="usepublish"></a>
<p>A hook for generating a function that publishes a view event. The callback function provided takes one argument, <code>data</code>, and returns an array containing three elements that describe the event to be published: 1. The scope to which the event is being emitted. 2. The name of the event. 3. The data attached to the event. The <code>data</code> argument corresponds to the value passed to the function returned by this hook.</p>
### Syntax
```typescript
function usePublish(callback: function(string): T): function(): void
```
### Parameters
<ParamField>
<ParamField path="callback" type="function(string): T" required={true}>
<p>The callback function used to construct the event data.</p>
</ParamField>
</ParamField>
### Returns
<ResponseField type="function(): void">
<p>The function to be used to publish the event.</p>
</ResponseField>
### Examples
<CodeGroup>
```typescript
type GrabData = { viewId: string, id: string };
const publishRelease = usePublish<GrabData>(
 (data) => [model.id, 'release', data]
);
// Call the generated function to publish the 'release' event with specific data
publishRelease({ viewId: myViewId, id });
```
</CodeGroup>
