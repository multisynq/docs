## InMultisynqSession Component

<a id="inmultisynqsession"></a>
<p>Main wrapper component that starts and manages a Multisynq session, enabling child elements to use the hooks described above. It takes the same parameters as [Session.join](../client/Session.html#.join) except that it doesn't need a root View class, since multisynq-react provides a suitable View class behind the scenes.</p>
### Usage
<CodeGroup>
```javascript
function MyApp() {
   return (
     <InMultisynqSession
       apiKey="1_123abc",
       appId="com.example.myapp"
       name="mySession"
       password="secret"
       model={MyRootModel}
        ...
     >
       // child elements that use hooks go here...
       <MyComponent/>
     </InMultisynqSession>
   );
}
```
</CodeGroup>
